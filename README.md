# Amazon Sales & Revenue Analysis Dashboard
This project presents an interactive Power BI dashboard built using Amazon's 2019 sales data. It provides insights into revenue, sales performance, top-selling products and key business KPIs through dynamic visualizations and DAX measures. 

<img width="1275" height="647" alt="image" src="https://github.com/user-attachments/assets/872180ae-8f54-4fb7-97ff-0bea93f88b6b" />

## Features
- 📊 Interactive Power BI Dashboard
- 📈 Monthly Sales & Revenue Trends
- 💰 KPI Cards (Sales, Revenue, Profit, Orders)
- 🛍️ Product Category Performance
- 📅 Time-Series Analysis (2019)
- ⚡ DAX Measures & Calculated Columns
- 🔍 Dynamic Filters and Slicers

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel (Data Preparation)
